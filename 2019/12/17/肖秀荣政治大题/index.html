肖秀荣政治大题

一、马哲

1.普遍性和特殊性

  1. 普遍性：矛盾具有普遍性，矛盾存在于一切事物之中，存在于一切事物发展过程的始终。
  2. 特殊性：矛盾具有特殊性，具体事务在其运动中的矛盾以及每一矛盾的各个方面都有其特点。要具体问题具体分析。
  3. 如实分析矛盾的特殊性，才能认清事物的本质和发展，才能采取正确的针对办法解决矛盾。

2.同一性和斗争性

  1. 同一性：矛盾双方相互贯通，相互依存的性质和趋势
  2. 斗争性：矛盾双方相互分离，相互排斥的性质和趋势。斗争性表现为：（1）斗争推动矛盾双方力量对比发生变化，造成事物的质变。（2）斗争促使矛盾双方地位或性质发生变化，造成事物质变。
  3. 同一性是相对的，有条件的；斗争性是无条件的，绝对的。
  4. 同一性和斗争性相结合，构成事物的矛盾运动，推动事物的变化发展.

3.斗争性

  1. 矛盾的斗争性有丰富的内容和多样的形式。不同的矛盾具有不同的斗争形式，同一矛盾在不同的发展阶段上斗争形式也不同。
  2. 矛盾分析法具有广泛而深刻的内容，把握矛盾普遍性与特殊性相统一方法，“两点论”与“重点论”相结合的方法，
  3. 矛盾具有特殊性，具体事物在其运动中的矛盾以及每一矛盾的各个方面各有其特点。要具体问题具体分析
  4. 事物由多种矛盾构成，事物的性质由主要矛盾的主要方面所规定的，因此必须在斗争中抓住主要矛盾，抓主要矛盾的主要方面。
  5. 矛盾的同一性和斗争性相互联结，相互制约因此必须在对立中把握同一，和在同一种把握对立。

4.质量

  1. 质是事物成为自生并区别于其他事物的内在规定性。
  2. 量是事物的规模，程度等可以用数量关系表示的内在规定性。
  3. 度是质和量的统一，是保持事物稳定性的数量界限，即事物的范围，程度。超出度的范围，此物就会转化成他物。XXX是事物发生质变的临界点。要坚持底线思维，防止“过”犹“不及”。

5.唯物史观–人民群众

  1. 唯物史观人为，在社会历史发展过程中，人民群众起着决定作用。人民群众是社会历史的实践的主题，是历史创造者。
  2. 人民群众是社会物质财富的创造者，是社会精神财富的创造者，是社会变革的决定力量。人民群众的总体意愿和行动代表了历史前进的方向，人民群众的社会实践最终决定历史发展的结局。

6.唯物史观–杰出人物

  1. 唯物史观从人民群众创造历史这一基本前提出发，即肯定了人民群众是社会历史的创造者，也不否认个人在历史上的作用。
  2. 按照个人对社会历史起作用的程度不同，区分为普通个人和历史人物，历史人物是一定历史事件的主要倡导者，组织领导或理论科学文化的重要代表人物。
  3. 杰出人物是历史人物中对推动历史社会发展做出重要贡献或起重要作用的人。在历史发展进程中，新的历史人物往往有具有进步意义的历史人物首先发现或提出。

7.理性认识，非理性认识，偶然和必然，实践

  1. 理性认识：人们借助抽象思维，在概括整理大量非理性材料的基础上，达到最事物本质，属性，内在联系和自身规律性的认识。理性认识是认识的高级阶段，是认识运动的第一次飞跃。

    理论的重要性在于能够指导实践，正确理性的认识能够指导实践，使实践顺利进行，达到预期的目的。

  2. 实践：实践决定认识，实践是认识的来源。人们只有通过实践，才能准确把握事物的本质属性和规律。

  3. 非理性认识：指人的情感，意志和认识能力值中具有非逻辑性的联想，猜想，顿悟等。非理性因素对人的认识能力具有：驱动，激活和控制作用。因此在认识过程中既要注重理性因素，也要主要非理性因素。

  4. 偶然和必然：事物发展有偶然和必然的方面。必然是事物一定发生，确定不移的性质和趋势，偶然是事物可能发生，可能不发生；可能这样发生，也可能那样发生的形式和趋势。

    必然存在于偶然之中，通过大量偶然表现出现并为自己开辟道路。

    偶然背后隐藏必然，是必然的表现和补充。

8.人与自然关系–顺应自然

  1. 自然界是人类社会的形成的自然基础，人类社会的存在和发展反过来作用于自然界。
  2. 实践是使物质世界分化为自然界和人类社会的历史前提，又是使自然界和人类社会相统一的现实基础。
  3. 人类在事件活动中按照自己的意愿改造世界的同时，要尊重自然规律。正确的事件的观点是理解人与自然关系，实现人与自然和谐相处的关键。

9.实践–发挥能动性

  1. 实践是人能动的改造客观世界的社会性的物质活动。实践具有自觉能动性，人的实践是一种有意识，有目的的活动，是以改造客观世界为目的的客观过程。
  2. 尊重客观规律是发挥主观能动性的前提。
  3. 只有充分发挥主观能动性，才能正确认识和利用客观规律。承认客观规律，并不是说人在客观规律面前是无能为力的。
  4. 人们通过自觉活动能够认识规律和利用规律，并按照客观规律去改造世界，满足自身需要。

10. 发展和新事物

  1. 发展是前进上升的运动，发展是实质是新事物的产生和旧事物的消亡。
  2. 新事物是符合历史前进方向，具有远大前途的东西，新事物是具有强大生命力，是不可战胜的。
  3. 新事物从旧事物中孕育出来，否定旧事物中腐朽落后的东西，保留了旧事物中仍合理的，符合新环境的因素，并且添加了旧事物所不能容纳的东西。
  4. 事物的发展不是一帆风顺的，是前进性和曲折性的统一，在总体上呈现出一种螺旋上升或波浪式前进的过程，

  2. 党的领导是中国特色社会主义制度的本质要求，是中国特色社会主义制度的最大优势。
