实体类定义规则

发表于 2020-03-02 | 分类于 技术杂文

简介

PO

  • persistent object 持久化对象
  • 对应数据库中的entity；

VO

  • view object 表示层对象
  • 用来对应页面显示的数据对象

DTO

  • Data Transfer object 数据传输对象

POJO

  • plain ordinary java object 无规则的java对象

    一个中间对象，可以转化为PO，VO，DTO

    • POJO持久化后==》PO
    • POJO用作表现层==》VO
    • POJO传输过程中==》DTO
